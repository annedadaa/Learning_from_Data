### Offensive Language Identification

How to run the code:

1) Clone this repository
2) You should run the all scripts the root directory

To train models:
```
python3 evaluate/evaluate.py --pred output/SVM/svm_preds.tsv --cf
```

To evaluate results:
```
python3 evaluate/evaluate.py --pred output/SVM/svm_preds.tsv --cf
```
